# 关于二次开发

**技术支持政策：**

本项目为开源项目，旨在为机器人爱好者和研究人员提供学习和参考。关于二次开发的技术支持，请注意以下说明：

- **文档优先：** 本技术手册已包含完整的原理说明、代码结构、API文档和开发指南。在提出技术问题前，请优先查阅本手册相关章节。
- **有限支持：** 项目维护者精力有限，无法提供一对一的详细技术指导。对于已购买用户的二次开发相关问题，我们可以提供建议和方向性指导，但回复可能较慢。
- **社区互助：** 建议通过GitHub Issues、QQ群（344319339）等渠道与其他开发者交流，共同解决问题。

**免责声明：**

- **硬件损坏：** 用户进行二次开发（包括但不限于修改代码、添加功能、调整参数等）可能导致硬件损坏、舵机烧毁、电路故障等问题。因二次开发导致的任何硬件损坏，我们不承担责任。
- **功能异常：** 修改代码或参数可能导致机器人运动异常、失控等风险。请在安全环境下进行测试，避免造成人身伤害或财产损失。

**建议：**

1. **充分理解：** 在进行二次开发前，请充分理解本手册中的原理介绍和代码结构。
2. **逐步测试：** 建议采用小步迭代的方式，每次修改后进行充分测试。
3. **安全第一：** 在安全、开阔的环境中进行测试，避免在危险环境下操作。
4. **查阅文档：** 遇到问题时，优先查阅本手册相关章节和代码注释。

**文档完善性：**

本技术手册已包含以下完整内容：
- 硬件配置和原理介绍
- 软件框架和代码结构
- 完整的API文档
- 动作组扩展开发教程
- 二次开发指南
- 故障排除和常见问题

绝大多数开发问题都可以通过查阅本手册得到解答。如确需技术支持，请提供：
- 已查阅的相关章节
- 具体的问题描述
- 错误日志或现象
- 已尝试的解决方法

感谢您的理解和支持！
