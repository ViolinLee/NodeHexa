# 机器人组装紧固件清单

## 组装所需紧固件说明

本项目现已支持**六足 / 四足双构型**。两种构型的机身件（上/下盖、PCB固定等）紧固件基本一致，但**与“腿/关节数量”直接相关**的紧固件数量会随构型变化。

说明：
- 下表中的“**可选连接**”项（如 `M2×12mm` 与对应螺母）仅在你采用该可选结构方案时才需要；若未使用可选结构，可从BOM中剔除。
- 数量为**不含备品**的建议采购数量，建议按实际装配习惯准备少量余量。

- 请按照规格要求准备相应数量的紧固件

- 轴销用于关节连接，确保运动顺畅
